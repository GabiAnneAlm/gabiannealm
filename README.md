**`Analista de Engenharia de Software QA`**
<br/>
<br/>


Sou estudante de Engenharia da Computação e Técnica em Eletrotécnica, com uma paixão por tecnologia, especialmente nas áreas de IoT e Sistemas Embarcados. Possuo cinco anos de experiência em desenvolvimento de hardware, abrangendo desde o layout de placas, prototipagem e montagem final, até a validação e criação de firmware. Atualmente, atuo como Analista de Engenharia na área de Engenharia de Software. Estou sempre em busca de aprimoramento para agregar valor como desenvolvedora, seja em firmware ou hardware voltado para IoT e Sistemas Embarcados.
